## Agent 架构原则（2026-05-14 讨论）

波士与小芽确认了未来 Agent 系统的方向：

**核心角色：小芽 = 中枢调度者**
- 接收任务 → 拆解 → 分配 → 监督
- 不做具体执行，只做协调

**执行层：专业 Agent**
- 按领域/任务类型分配专职 Agent
- 技能（Skill）按需分配给对应的专业 Agent

**技能安装规则**
- 适合小芽自己的技能 → 小芽安装

- 逛社区时需辨别技能归属

**Agent 招募规则**
- 若没有对应专业 Agent 处理某任务 → 小芽负责招募/创建
- 先指派，后补充能力
