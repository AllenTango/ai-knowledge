# 知识库推荐技能

从 `knowledge/articles/` 目录中读取知识条目，按评分排序后推荐给用户。

## 使用场景

- 用户说"推荐文章"、"给我高分文章"、"最佳知识"、"最高分的条目"
- 用户想要"最好的"、"评分最高的"、"最值得看的"知识

## 推荐逻辑

1. 扫描 `knowledge/articles/` 目录下的所有 `.json` 文件
2. 按 `score` 字段降序排序
3. 返回 Top N 条目（默认 5 条，最多 10 条）

## 执行脚本

```bash
python3 .openclaw/knowledge-recommend/scripts/recommend.py [--top N] [--min-score SCORE]
```

参数:
- `--top N`: 返回前 N 条，默认 5，最大 10
- `--min-score SCORE`: 最低评分阈值（0.0-10.0），默认 0

## 输出格式

返回 Markdown 格式的推荐列表：

```
📚 知识推荐 Top 5

🟢 9.2分 | 文章标题
   摘要...
   标签: #机器学习 #Python
   来源: GitHub

🟡 7.8分 | 文章标题
   ...
```

## 评分等级

| 评分 | 标识 | 含义 |
|------|------|------|
| 8.0+ | 🟢 | 强烈推荐 |
| 6.0-7.9 | 🟡 | 值得一看 |
| 4.0-5.9 | 🟠 | 一般 |
| <4.0 | 🔴 | 忽略 |

## 注意事项

- 只返回有实际内容的条目（跳过 score=0 或无摘要的）
- 按需设置 `--min-score` 过滤低质量内容
- 如果知识库为空，返回"暂无知识条目"的提示
